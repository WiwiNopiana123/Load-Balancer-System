#####> Load-Balancer-System
- Untuk konfigurasi ip adress sistem:
sudo nano /etc/nginx/sites-available/roundrobin-lb

- Untuk aktifkan virtual envirotment:
source venv/bin/activate

- Untuk pindah ke folder aplikasi:
cd ~/roundrobin-lb

- Untuk cek ip adress Ubuntu:
ip a

- untuk cek ip adress Windows:
ipconfig

- Untuk menjalankan aplikasi menggunakan flask:
python3 [nama_app].py - contoh: python3 agent1.py

- Cek NGINX:
sudo lsof -i :8000
systemctl status nginx
curl -I http://192.168.1.72:8000/
